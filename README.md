# Delicake Confection Digital - E-Commerce Platform

A modern, responsive, and user-centric web application designed and built for **Delicake Confection Digital**, a premium bespoke bakery business operating in Zimbabwe. This platform allows customers to browse custom artisanal cakes and cupcakes, create profiles, securely place orders, and track deliveries straight to their desired destination.

---

## 📌 Project Overview

This project was developed as a digital storefront solution to transform Delicake Confection Digital's offline bakery operations into a fully automated e-commerce experience. The design heavily features the brand's signature luxury gold-on-black aesthetic.

### Key Client Requirements Integrated:
* **Target Market:** Purpose-built for functions, birthdays, anniversaries, and holiday celebrations (Mother's Day, Christmas, public holidays) within Zimbabwe.
* **Operational Rules:** Explicitly enforces the brand's policy requiring a **50% non-refundable deposit** to confirm any baking slots.
* **Interactive Features:** End-user profiles, secure login pipelines, order placement engines, and an integrated destination delivery/order-tracking workflow.

---

## ✨ Features

### 1. User Authentication & Profile Management
* **Client Profile Creation:** Users can register accounts to manage their personal details.
* **Order History & Tracking:** A dedicated dashboard page where users can monitor live progress ("In the Oven", "Decorating", "Out for Delivery") and target addresses.

### 2. Digital Product Showcase & E-Commerce
* **Bespoke Categorization:** Clean architectural divisions separating services for *Weddings*, *Birthdays*, and *Holidays*.
* **Dynamic Visual Gallery:** A high-definition responsive layout exhibiting the bakery's unique design capabilities (including character cakes, drip cakes, and custom typography toppers).
* **Cart System:** "Add to Cart" functionality handling customized order parameters.

### 3. Business Guardrails & Communications
* **Strict Deposit Notice:** Global warning bars and cart notices alerting users of the mandatory 50% deposit rule.
* **Lead-Capture Contact Form:** Asynchronous email/database submission form for handling custom event inquiries.
* **Localized Operational Settings:** Displays precise trading hours (Monday, Tuesday, Thursday, Friday, Saturday from 9:00 AM – 4:00 PM) and Zimbabwe specific contact paths (+263 775805912).

---

## 🛠️ Technology Stack Used

* **Frontend:** Semantic HTML5, Advanced CSS3 Grid & Flexbox mechanics (mobile-first layout design), JavaScript (ES6+) for interactive UI elements.
* **Assets:** Optimized Base64 image encoding and compressed image assets to ensure lightning-fast performance over localized mobile networks.
